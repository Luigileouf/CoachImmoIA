# Design Decisions

Ce dossier sert a consigner les decisions design importantes quand elles deviennent structurantes pour le produit.

Exemples :

- changement de direction artistique
- nouveaux principes de navigation
- arbitrage entre `Assistant IA` et `Projet`
- evolution du role du `Coach humain`
- systeme de composants retenu

## Format recommande

Un fichier par decision, par exemple :

- `2026-05-14-direction-artistique.md`
- `2026-05-14-sitemap-produit.md`

Chaque note doit idealement contenir :

- contexte
- decision
- impact produit
- impact front
- questions ouvertes
